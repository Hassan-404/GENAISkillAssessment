# Voice-Enabled E-Commerce Customer Service Agent

A voice-enabled AI agent for order tracking and return processing, powered by Whisper (speech-to-text), LangChain (agent framework), and RAG (policy retrieval).

## Features
- **Voice & Text Input**: Process customer queries via microphone or text
- **Order Tracking**: Check status using order ID or email
- **Return Processing**: Multi-step workflow with policy verification
- **RAG Integration**: FAISS vector store for policy document retrieval
